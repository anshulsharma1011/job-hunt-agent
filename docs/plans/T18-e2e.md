- MongoDB running on `localhost:27017`

## `tests/e2e/test_smoke.py`

```python
import pytest
from pathlib import Path

pytestmark = pytest.mark.e2e    # skip with: pytest -m "not e2e"

def test_full_cycle_smoke(e2e_config, tmp_path):
    # 1. Setup
    deps = build_deps(e2e_config)
    deps.reporter_agent = ReporterAgent(tmp_path)
    graph = build_graph(deps)

    # 2. Run profile setup first
    pdf = Path("materials/resume.pdf")
    profile, criteria = deps.profile_agent.run(pdf)
    deps.profile_repo.save(profile)

    # 3. Run full cycle
    state = make_initial_state(cycle_id="cycle_smoke_001")
    result = graph.invoke(state)

    # 4. Assert report written
    report_path = Path(result["report_path"])
    assert report_path.exists()
    assert report_path.stat().st_size > 0

    # 5. Assert MongoDB has data
    opps = deps.opportunity_repo.get_by_cycle("cycle_smoke_001")
    assert len(opps) > 0

    cycle = deps.cycle_repo.get_latest()
    assert cycle is not None
    assert cycle.completed_at is not None
    assert cycle.discovered_count > 0
```

## Running E2E Tests

## Steps

1. Place your resume PDF at `materials/resume.pdf`
2. Ensure Ollama is running: `curl http://localhost:11434/` → "Ollama is running"
3. Ensure MongoDB is running: `brew services list | grep mongodb`
4. Write `tests/e2e/conftest.py` and `tests/e2e/test_smoke.py`
5. Run `pytest tests/e2e/ -m e2e -v`
6. Inspect `output/` for the generated report
7. Commit
